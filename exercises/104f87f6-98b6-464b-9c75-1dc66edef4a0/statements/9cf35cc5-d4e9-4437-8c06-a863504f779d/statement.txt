        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Você terá que usar métodos da classe String para manipular strings. </font><font style="vertical-align: inherit;">Você pode muito bem ter que usar os seguintes métodos:</font></font></p> 
        <ul> 
         <li><code>length()</code></li>

        <h3><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ENTRADA</font></font></strong></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">A entrada começa com um número indicando quantos casos de teste serão processados. </font><font style="vertical-align: inherit;">Cada um dos casos estará em uma linha diferente e será composto por uma string de texto não vazia.</font></font></p> 
        <h3><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">SAÍDA</font></font></strong></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Para cada caso de teste, o programa escreverá a frase resultante da inversão da string.</font></font></p> 
        <h3><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">RESTRIÇÕES</font></font></strong></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">O problema deve ser resolvido projetando uma função que receba uma string de texto e retorne a string invertida.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">O programa principal, para cada um dos casos, lerá os dados de entrada e utilizará a função para resolver o problema, imprimindo posteriormente o resultado correspondente.</font></font></p></body></html>

        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Queremos fazer um programa Java que conte o número de vogais nas frases que são inseridas.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Lembre-se de que as frases podem conter letras maiúsculas e minúsculas e, em ambos os casos, as vogais devem ser contadas.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Você terá que usar métodos da classe String para manipular strings. </font><font style="vertical-align: inherit;">Você pode muito bem ter que usar os seguintes métodos:</font></font></p> 
        <ul> 
         <li><code>charAt(n)</code></li> 
         <li><code>toLowerCase()</code></li> 
         <li><code>length()</code></li> 
        </ul> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Além disso, para ler os dados da entrada, talvez seja necessário usar os seguintes métodos da classe Scanner:</font></font></p> 
        <ul> 
         <li><code>nextInt()</code></li> 
         <li><code>nextLine()</code></li> 
        </ul> 
        <h3><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ENTRADA</font></font></strong></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">A entrada começa com um número indicando quantos casos de teste serão processados.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Cada um dos casos é composto por uma frase (uma sequência de texto)</font></font></p> 
        <h3><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">SAÍDA</font></font></strong></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Para cada caso de teste, o programa escreverá o número de vogais que a frase correspondente possui.</font></font></p> 
        <h3><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">RESTRIÇÕES</font></font></strong></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">O problema deve ser resolvido projetando uma função que receba uma string de texto e retorne o número de vogais que ela contém.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">O programa principal, para cada um dos casos, solicitará as strings pelo teclado e utilizará a função para solucionar o problema, imprimindo posteriormente o resultado correspondente.</font></font></p></body></html>

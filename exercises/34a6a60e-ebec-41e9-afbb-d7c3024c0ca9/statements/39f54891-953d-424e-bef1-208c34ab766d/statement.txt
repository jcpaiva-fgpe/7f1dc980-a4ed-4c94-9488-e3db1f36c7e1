        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">For these damages, the state government has established 3 types of aid:</font></font></p> 
        <ol> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">For total destruction of the habitual residence</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">For damages that affect the structure of the habitual residence</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">For damages that do not affect the structure of the habitual residence</font></font></li> 
        </ol> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Therefore, the first thing the application should do is display a menu asking the user what type of help they are requesting, or 0 if they just want the program to end.</font></font><small><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Copy and paste the OUTPUT texts, so that there are no typographical errors in the menu.</font></font></em></small></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">This menu will be displayed through the function </font></font><code>mostrarMenu()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">If the user chooses any of the 3 types of help, the application must ask the user for the </font></font><code>"Cantidad solicitada: "</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">. </font><font style="vertical-align: inherit;">The amount that the user requests may not exceed the maximum amounts of aid established in </font></font><a href="https://translate.google.com/website?sl=es&amp;tl=en&amp;hl=es&amp;client=webapp&amp;u=https://www.boe.es/buscar/act.php?id%3DBOE-A-2021-16231"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Royal Decree-Law 20/2021</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> and which are shown below for each of the types of aid:</font></font></p> 
        <ol> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">For total destruction of the habitual residence: </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">30240</font></font></strong></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">For damages that affect the structure of the habitual residence: </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">20640</font></font></strong></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">For damages that do not affect the structure of the habitual residence: </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">10320</font></font></strong></li> 
        </ol> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">For this reason, the application must call the function </font></font><code>cuantiaOtorgada()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, to which the type of help and the amount requested is sent. </font><font style="vertical-align: inherit;">The function </font></font><code>cuantiaOtorgada()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">will return:</font></font></p> 
        <ul> 
         <li><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">the amount requested</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> , if it is </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">not</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> higher than the maximum amount for that type of aid. </font></font><br><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">either</font></font></li> 
         <li><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">the maximum amount established for the type of aid</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> , in the event that the amount requested is greater than the maximum for that type of aid.</font></font></li> 
        </ul> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Finally, when the user decides to exit the application ( </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">choosing option 0 from the menu)</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> , the application must report the total amount of aid granted.</font></font></p></body></html>

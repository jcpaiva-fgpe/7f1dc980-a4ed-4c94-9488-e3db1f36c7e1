        <h3><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">INTRÄDE</font></font></strong></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Posten börjar med ett nummer som anger hur många testfall som kommer att behandlas.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vart och ett av fallen kommer att ligga på en rad med följande format:</font></font></p> 
        <ul> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Ett heltal som anger storleken på kvadratens sidor. </font><font style="vertical-align: inherit;">Om det angivna siffran är mindre än 2, kommer rektangeln inte att målas.</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Ett JA eller NEJ för att indikera om kanten på kvadraten ska målas. </font><font style="vertical-align: inherit;">Om det är JA, följer tecknet som ska användas för att måla gränsen.</font></font></li> 
         <li>Un SI o un NO para indicar si se pinta el relleno o interior del cuadrado. Si es un SI, entonces a continuación vendrá el caracter que hay que utilizar para pintar el relleno o interior.</li> 
        </ul> 
        <h3><strong>SALIDA</strong></h3> 
        <p>Por cada caso de prueba, el programa pintará el cuadrado correspondiente.</p> 
        <p>Entre caso y caso se imprimirá CASO y el número del caso.</p> 
        <h3><strong>RESTRICCIONES</strong></h3> 
        <p>Deberá resolverse el problema diseñando una función (técnicamente un PROCEDIMIENTO)&nbsp;que reciba como entrada el tamaño del lado del cuadrado y el caracter que debe usarse para pintar el borde del cuadrado y el caracter que debe usarse para pintar el relleno del cuadrado. El procedimiento dibujará por pantalla el cuadrado del tamaño correspondiente usando los caracteres indicados.</p>

        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Exempel:</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vi kan skapa flera instanser av den namngivna klassen </font></font><code>SubClase</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">:</font></font></p> 
        <blockquote> 
         <p><code>package oop;</code></p> 
         <p><code>public class OOP {<br> <br> &nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <strong>SubClase obj1 = new SubClase();</strong></code><br> <code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <strong>SubClase obj2 = new SubClase();</strong></code><br> <code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println(<strong>obj1.x</strong>);</code><br> <code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println(<strong>obj2.x</strong>);</code><br> <code>&nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
         <p><code>class SubClase {<br> &nbsp; int x = 5;<br> }</code></p> 
        </blockquote> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Träning:</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Skapa två objekt av Modulo-klassen, som du utvecklade i föregående övning, och skriv, från main()-metoden, värdet på de två anteckningarna.</font></font></p> 
        <p><small><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Notera:</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> För tillfället kommer värdet på noten för alla instanser som skapas av Modulo-klassen att vara detsamma. </font><font style="vertical-align: inherit;">Senare kommer vi att se hur man tilldelar olika betyg till varje modul.</font></font></small></p></body></html>

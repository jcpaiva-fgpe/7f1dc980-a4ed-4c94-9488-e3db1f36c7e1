        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">I programmering kallas </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">handlingen att skapa en variabel</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">DEFINIERA en variabel</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> " eller " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">DEKLÄRA en variabel</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> ". </font><font style="vertical-align: inherit;">När en variabel deklareras är </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">dess minnesutrymme reserverat</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> . </font><font style="vertical-align: inherit;">I de flesta programmeringsspråk finns det specifika instruktioner för att göra denna variabeldefinition eller deklaration. </font><font style="vertical-align: inherit;">När vi definierar en variabel ger vi den ett namn. </font><font style="vertical-align: inherit;">Detta namn kan inte ändras och det kan inte finnas två variabler med samma namn i samma program (förutom om de är från olika scope).</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Följande kod deklarerar en namngiven variabel </font></font><code>horasModulo</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">av typen </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">int</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> :</font></font></p>

         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">I programmering kallas satsen som används för att ställa in en variabel en </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ASSIGN-sats</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> , och åtgärden kallas " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">tilldela ett värde till en variabel</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> ".</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vi kommer att lägga till, till den föregående koden, en tilldelningssats, för att tilldela värdet </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">7</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> till </font></font><code>horasModulo</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">.</font></font></p> 
        <blockquote> 
         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; horasModulo = 7;<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vi måste alltid </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">tilldela ett initialvärde till variabler i våra program</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> . </font><font style="vertical-align: inherit;">Detta är vad vi kallar </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">INITIALISERA</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> en variabel eller </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">INITIALISERA</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> en variabel.</font></font></p> 
        <p>Una vez que la variable ha sido definida e inicializada, ya puede ser usada en nuestro programa, bien leyendo su contenido (usando su valor), bien dándole un nuevo valor mediante una nueva sentencia de asignación. El valor de una variable puede cambiar muchas veces a lo largo de un programa. Estas operaciones se llaman<strong> leer el valor de una variable</strong> o <strong>escribir el valor de una variable</strong> (mediante una <strong>sentencia de asignación</strong>).</p> 
        <p>En el siguiente código, vamos a leer el valor de la variable <code>horasModulo </code>para concatenarlo (con el operador +) a la cadena de caracteres "Las horas del módulo de Programación son: ", obteniendo el texto <em><code>Las horas del módulo de Programación son: 7</code></em>.</p> 
        <blockquote> 
         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {</code><br> <code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; horasModulo = 7;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println("Las horas del módulo de Programación son: " + horasModulo);<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
        </blockquote>

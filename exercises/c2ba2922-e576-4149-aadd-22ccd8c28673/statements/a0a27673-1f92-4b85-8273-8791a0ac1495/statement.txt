        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">In programming, the </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">action of creating a variable</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> is called " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">DEFINE a variable</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> " or " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">DECLARE a variable</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> ". </font><font style="vertical-align: inherit;">When a variable is declared, </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">its memory space is reserved</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> . </font><font style="vertical-align: inherit;">In most programming languages, there are specific instructions for doing this variable definition or declaration. </font><font style="vertical-align: inherit;">When we define a variable, we give it a name. </font><font style="vertical-align: inherit;">This name cannot be changed and there cannot be two variables with the same name in the same program (except if they are from different scopes).</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">The following code declares a named variable </font></font><code>horasModulo</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">of type </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">int</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> :</font></font></p>

         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">In programming, the statement used to set a variable is called an </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ASSIGN statement</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> , and the action is called " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">assign a value to a variable</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> ".</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">We are going to add, to the previous code, an assignment statement, to assign the value </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">7</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> to </font></font><code>horasModulo</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">.</font></font></p> 
        <blockquote> 
         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; horasModulo = 7;<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">We must always </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">assign an initial value to variables in our programs</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> . </font><font style="vertical-align: inherit;">This is what we call </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">INITIALIZING</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> a variable or </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">INITIALIZING</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> a variable.</font></font></p> 
        <p>Una vez que la variable ha sido definida e inicializada, ya puede ser usada en nuestro programa, bien leyendo su contenido (usando su valor), bien dándole un nuevo valor mediante una nueva sentencia de asignación. El valor de una variable puede cambiar muchas veces a lo largo de un programa. Estas operaciones se llaman<strong> leer el valor de una variable</strong> o <strong>escribir el valor de una variable</strong> (mediante una <strong>sentencia de asignación</strong>).</p> 
        <p>En el siguiente código, vamos a leer el valor de la variable <code>horasModulo </code>para concatenarlo (con el operador +) a la cadena de caracteres "Las horas del módulo de Programación son: ", obteniendo el texto <em><code>Las horas del módulo de Programación son: 7</code></em>.</p> 
        <blockquote> 
         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {</code><br> <code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; horasModulo = 7;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println("Las horas del módulo de Programación son: " + horasModulo);<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p>

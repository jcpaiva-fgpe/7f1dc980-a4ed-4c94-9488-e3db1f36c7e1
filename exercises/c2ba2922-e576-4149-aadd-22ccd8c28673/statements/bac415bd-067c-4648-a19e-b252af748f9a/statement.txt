        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Na programação, a </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ação de criar uma variável</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> chama-se " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">DEFINE uma variável</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> " ou " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">DECLARE uma variável</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> ". </font><font style="vertical-align: inherit;">Quando uma variável é declarada, </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">seu espaço de memória é reservado</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> . </font><font style="vertical-align: inherit;">Na maioria das linguagens de programação, existem instruções específicas para fazer essa definição ou declaração de variável. </font><font style="vertical-align: inherit;">Quando definimos uma variável, damos-lhe um nome. </font><font style="vertical-align: inherit;">Este nome não pode ser alterado e não pode haver duas variáveis ​​com o mesmo nome no mesmo programa (exceto se forem de escopos diferentes).</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">O código a seguir declara uma variável nomeada </font></font><code>horasModulo</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">do tipo </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">int</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> :</font></font></p> 
        <blockquote> 
         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Na programação, a instrução usada para definir uma variável é chamada de </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">instrução ASSIGN</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> e a ação é chamada de " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">atribuir um valor a uma variável</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> ".</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vamos adicionar, ao código anterior, um comando de atribuição, para atribuir o valor </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">7</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> a </font></font><code>horasModulo</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">.</font></font></p> 
        <blockquote> 
         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; horasModulo = 7;<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Devemos sempre </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">atribuir um valor inicial às variáveis ​​em nossos programas</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> . </font><font style="vertical-align: inherit;">Isso é o que chamamos de </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">INICIALIZAR</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> uma variável ou </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">INICIALIZAR</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> uma variável.</font></font></p> 
        <p>Una vez que la variable ha sido definida e inicializada, ya puede ser usada en nuestro programa, bien leyendo su contenido (usando su valor), bien dándole un nuevo valor mediante una nueva sentencia de asignación. El valor de una variable puede cambiar muchas veces a lo largo de un programa. Estas operaciones se llaman<strong> leer el valor de una variable</strong> o <strong>escribir el valor de una variable</strong> (mediante una <strong>sentencia de asignación</strong>).</p> 
        <p>En el siguiente código, vamos a leer el valor de la variable <code>horasModulo </code>para concatenarlo (con el operador +) a la cadena de caracteres "Las horas del módulo de Programación son: ", obteniendo el texto <em><code>Las horas del módulo de Programación son: 7</code></em>.</p> 
        <blockquote> 
         <p><code>package variables;</code></p> 
         <p><code>public class Variables {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {</code><br> <code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int horasModulo;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; horasModulo = 7;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println("Las horas del módulo de Programación son: " + horasModulo);<br> &nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp; &nbsp;<br> }</code></p> 
        </blockquote>

        <ul> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">deklarera egenskaperna för klassen som </font></font><code>private</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">.</font></font></li> 
         <li><font style="vertical-align: inherit;"></font><code>public</code> <strong><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">utveckla hämta</font></font></em></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> och </font></font><strong><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ställa in</font></font></em></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> metoder </font><font style="vertical-align: inherit;">för att komma åt och uppdatera värdet på varje deklarerad egenskap </font></font><code>private</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">.</font></font></li> 
        </ul> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Fördel</font></font></h3> 
        <ul> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Bättre kontroll över egenskaper och metoder.</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Flexibilitet när det gäller att modifiera en del av koden utan att påverka andra delar.</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Ökar datasäkerheten.</font></font></li> 
        </ul> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Hämta och ställ in</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vi lärde oss att fastigheter </font></font><code>private</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">endast kan nås inom samma klass. </font><font style="vertical-align: inherit;">De kan dock nås genom </font></font><em><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">get-</font></font></strong></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> och </font></font><strong><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">set</font></font></em></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> -metoder som deklarerats som </font></font><code>public</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">En metod </font></font><code>get</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">returnerar variabelns värde och metoden kommer att </font></font><code>set</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ändra dess värde.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Syntaxen för båda metoderna är densamma: de börjar med </font></font><code>get</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">eller </font></font><code>set</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">följs av egenskapsnamnet med den första bokstaven versal.</font></font></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Exempel:</font></font></h3> 
        <blockquote> 
         <p><code>public class OOP_ejemplos {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; SubClase miObj = new SubClase(4, 3);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println(miObj.suma());<br> &nbsp;&nbsp;&nbsp; }</code></p> 
         <p><code>}</code></p> 
         <p><code>class SubClase {</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; private int x, y;<br> &nbsp;&nbsp; &nbsp;<br> &nbsp;&nbsp;&nbsp; public SubClase(int _x, int _y) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; setX(_x);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; setY(_y);<br> &nbsp;&nbsp;&nbsp; }</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public int getX() {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; return x;<br> &nbsp;&nbsp;&nbsp; }</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public int getY() {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; return y;<br> &nbsp;&nbsp;&nbsp; }</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public void setX(int x) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; this.x = x;<br> &nbsp;&nbsp;&nbsp; }</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public void setY(int y) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; this.y = y;<br> &nbsp;&nbsp;&nbsp; }</code></p> 
         <p><code>&nbsp;&nbsp;&nbsp; public int suma() {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; return getX() + getY();<br> &nbsp;&nbsp;&nbsp; }<br> }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Du kan se en metod </font></font><code>get</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">och en annan metod </font></font><code>set</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">för var och en av egenskaperna.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Nyckelordet detta används för att referera till objektet med vilket metoden anropades.</font></font></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Träning</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Med utgångspunkt från klassen </font></font><code>Modulo</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">som löser den sista Modifiers-övningen, kapslar den in egenskaperna och genererar </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">get-</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> och </font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">set</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> -metoderna som motsvarar dessa egenskaper.</font></font></p>

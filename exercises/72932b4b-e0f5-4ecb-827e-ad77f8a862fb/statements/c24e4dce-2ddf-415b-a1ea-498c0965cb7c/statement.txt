        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">A estrutura </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">if</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> é uma estrutura de seleção, na qual uma expressão lógica é avaliada e, dependendo do resultado, uma instrução ou um bloco de instruções é executado.</font></font></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Sintaxe:</font></font></h3> 
        <blockquote> 
         <p><code>if (expresión-lógica) {<br> &nbsp;&nbsp;&nbsp; sentencia1;<br> &nbsp;&nbsp;&nbsp; sentencia2;<br> &nbsp;&nbsp;&nbsp; …;<br> &nbsp;&nbsp;&nbsp; sentenciaN;<br> }</code></p> 
        </blockquote> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Funcionamento</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Se a expressão lógica for avaliada como verdadeira, o bloco de instrução associado será executado. </font><font style="vertical-align: inherit;">Se o resultado desta avaliação for falso, nenhuma instrução associada à estrutura condicional será executada.</font></font></p> 
        <p><em><small><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Embora, no caso de uma única instrução ter que ser executada, seja possível evitar o uso de chaves, é aconselhável sempre incluir as chaves, mesmo que seja para incluir uma única instrução.</font></font></small></em></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Exemplo:</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">No exemplo a seguir, o texto 'APPROVED' será escrito se o valor atribuído à variável note for maior ou igual a 5.</font></font></p> 
        <blockquote> 
         <p><code>...<br> int nota = 7;<br> if(nota &gt;= 5) {<br> &nbsp;&nbsp;&nbsp; System.out.println("APROBADO");<br> }<br> ...</code></p> 
        </blockquote> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Exercício:</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Desenvolva um aplicativo que leia 2 notas do teclado e, para cada uma delas, escreverá o texto "APROVADO" apenas caso a nota digitada seja maior ou igual a 5.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Independentemente de termos escrito o texto "APPROVED" ou não, após cada uma das qualificações deve aparecer uma string como esta "========", conforme mostrado no OUTPUT.</font></font></p></body></html>

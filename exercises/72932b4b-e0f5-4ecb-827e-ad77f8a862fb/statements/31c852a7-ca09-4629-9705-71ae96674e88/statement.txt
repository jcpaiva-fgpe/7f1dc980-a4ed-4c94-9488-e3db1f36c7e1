        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">The </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">if</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> structure is a selection structure, in which a logical expression is evaluated and, depending on the result, a statement or a block of statements is executed.</font></font></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Syntax:</font></font></h3> 
        <blockquote> 
         <p><code>if (expresión-lógica) {<br> &nbsp;&nbsp;&nbsp; sentencia1;<br> &nbsp;&nbsp;&nbsp; sentencia2;<br> &nbsp;&nbsp;&nbsp; …;<br> &nbsp;&nbsp;&nbsp; sentenciaN;<br> }</code></p> 
        </blockquote> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Functioning</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">If the logical-expression evaluates to true, the associated statement block is executed. </font><font style="vertical-align: inherit;">If the result of this evaluation is false, no instruction associated with the conditional structure will be executed.</font></font></p> 
        <p><em><small><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Although, in the event that a single statement has to be executed, it is possible to avoid the use of braces, it is advisable to always include the braces, even if it is to enclose a single statement.</font></font></small></em></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Example:</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">In the following example, the text 'APPROVED' will be written if the value assigned to the note variable is greater than or equal to 5.</font></font></p> 
        <blockquote> 
         <p><code>...<br> int nota = 7;<br> if(nota &gt;= 5) {<br> &nbsp;&nbsp;&nbsp; System.out.println("APROBADO");<br> }<br> ...</code></p> 
        </blockquote>

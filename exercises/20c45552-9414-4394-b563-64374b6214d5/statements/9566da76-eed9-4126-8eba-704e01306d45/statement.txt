<p><strong>Implementa el método</strong><br>
<tt>&nbsp; private static void nuevoAcogido()</tt><strong>.<br>
Este método:</strong></p>

<ul>
	<li>
	<p>Insertará, en el array <tt>acogidos[],</tt> el nombre de un nuevo acogido introducido por teclado.</p>
	</li>
	<li>
	<p>La posición en la que se debe insertar será la que indique la variable <tt>numAcogidos</tt>. Esta variable se deberá incrementar por cada acogido almacenado en el array <tt>acogidos[]</tt>.</p>
	</li>
	<li>
	<p>Tanto el array <tt>acogidos[]</tt> como la variable <tt>numAcogidos</tt> deberán ser visibles en todos los métodos de la clase <tt>Acogimientos</tt>. El tamaño del array será el indicado por la constante&nbsp;<code>NUMERO_ACOGIDOS</code>.</p>
	</li>
	<li>
	<p>Hay que tener en cuenta que la capacidad máxima inicial del array es la que indica la constante <tt>NUMERO_ACOGIDOS</tt> y que hay que controlar la excepción <tt>ArrayIndexOutOfBoundsException</tt> en el caso de que se quieran almacenar más acogidos de los que permite el array <tt>acogidos[]</tt>.</p>
	</li>
</ul></body></html>

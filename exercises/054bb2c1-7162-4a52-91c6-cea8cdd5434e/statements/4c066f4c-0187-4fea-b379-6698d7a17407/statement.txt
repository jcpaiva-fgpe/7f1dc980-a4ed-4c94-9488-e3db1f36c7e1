        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">On many occasions, methods return some value as a result of their execution. </font><font style="vertical-align: inherit;">That value can be stored in some variable or used directly as a substitute for the invocation.</font></font></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Example</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">We are going to translate the code corresponding to the user's wish to continue or not to a function </font></font><code>elUsuarioQuieraSeguir()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, so that method returns the value typed by the user.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">The most important change in that method is that the modifier </font></font><code>void</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">must be replaced by the data type of the return value. </font><font style="vertical-align: inherit;">At this moment, the return value will be the number typed by the user, so the return value will be </font></font><code>int</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">:</font></font></p> 
        <blockquote> 
         <p><code>&nbsp;&nbsp;&nbsp; static <strong>int</strong> elUsuarioQuieraSeguir() {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int seguir;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Scanner sc = new Scanner(System.in);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; seguir = sc.nextInt();<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; return seguir;<br> &nbsp;&nbsp;&nbsp; }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">The code of the main() method would look like this:</font></font></p> 
        <blockquote> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int contador = 1, seguir = 1;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; while (<strong>elUsuarioQuieraSeguir() != 0</strong>) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // Invocamos el método cabecera<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; cabeceraTablaMultiplicar(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; tablaDeUnNumero(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // Invocamos el método pie<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; pieTablaMultiplicar(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; contador++;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp;&nbsp; }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Exercise</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Transform the function </font></font><code>elUsuarioQuieraSeguir()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">so that it returns a </font></font><code>boolean</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, instead of </font></font><code>int</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">: if the user presses a number other than 0, the function will return </font></font><code>true</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">and return </font></font><code>false</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">if the user has pressed a 0.</font></font></p></body></html>

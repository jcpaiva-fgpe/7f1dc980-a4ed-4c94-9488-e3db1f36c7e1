        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vid många tillfällen ger metoder ett visst värde som ett resultat av att de utförs. </font><font style="vertical-align: inherit;">Det värdet kan lagras i någon variabel eller användas direkt som en ersättning för anropet.</font></font></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Exempel</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vi kommer att översätta koden som motsvarar användarens önskan att fortsätta eller inte till en funktion </font></font><code>elUsuarioQuieraSeguir()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, så att metoden returnerar det värde som användaren har skrivit.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Den viktigaste förändringen i den metoden är att modifieraren </font></font><code>void</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">måste ersättas av returvärdets datatyp. </font><font style="vertical-align: inherit;">För närvarande kommer returvärdet att vara det nummer som angetts av användaren, så returvärdet blir </font></font><code>int</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">:</font></font></p> 
        <blockquote> 
         <p><code>&nbsp;&nbsp;&nbsp; static <strong>int</strong> elUsuarioQuieraSeguir() {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int seguir;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Scanner sc = new Scanner(System.in);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; seguir = sc.nextInt();<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; return seguir;<br> &nbsp;&nbsp;&nbsp; }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Koden för metoden main() skulle se ut så här:</font></font></p> 
        <blockquote> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int contador = 1, seguir = 1;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; while (<strong>elUsuarioQuieraSeguir() != 0</strong>) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // Invocamos el método cabecera<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; cabeceraTablaMultiplicar(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; tablaDeUnNumero(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // Invocamos el método pie<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; pieTablaMultiplicar(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; contador++;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp;&nbsp; }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Träning</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Omvandla funktionen </font></font><code>elUsuarioQuieraSeguir()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">så att den returnerar en </font></font><code>boolean</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, istället för </font></font><code>int</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">: om användaren trycker på en annan siffra än 0, kommer funktionen att returnera </font></font><code>true</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">och returnera </font></font><code>false</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">om användaren har tryckt på en 0.</font></font></p></body></html>

        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Em muitas ocasiões, os métodos retornam algum valor como resultado de sua execução. </font><font style="vertical-align: inherit;">Esse valor pode ser armazenado em alguma variável ou usado diretamente como substituto da invocação.</font></font></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Exemplo</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vamos traduzir o código correspondente ao desejo do usuário de continuar ou não para uma função </font></font><code>elUsuarioQuieraSeguir()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, para que o método retorne o valor digitado pelo usuário.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">A mudança mais importante nesse método é que o modificador </font></font><code>void</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">deve ser substituído pelo tipo de dados do valor de retorno. </font><font style="vertical-align: inherit;">Neste momento, o valor de retorno será o número digitado pelo usuário, então o valor de retorno será </font></font><code>int</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">:</font></font></p> 
        <blockquote> 
         <p><code>&nbsp;&nbsp;&nbsp; static <strong>int</strong> elUsuarioQuieraSeguir() {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int seguir;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Scanner sc = new Scanner(System.in);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; seguir = sc.nextInt();<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; return seguir;<br> &nbsp;&nbsp;&nbsp; }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">O código do método main() ficaria assim:</font></font></p> 
        <blockquote> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int contador = 1, seguir = 1;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; while (<strong>elUsuarioQuieraSeguir() != 0</strong>) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // Invocamos el método cabecera<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; cabeceraTablaMultiplicar(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; tablaDeUnNumero(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // Invocamos el método pie<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; pieTablaMultiplicar(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; contador++;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp;&nbsp; }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Exercício</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Transforme a função </font></font><code>elUsuarioQuieraSeguir()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">para que ela retorne a </font></font><code>boolean</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, em vez de </font></font><code>int</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">: se o usuário pressionar um número diferente de 0, a função retornará </font></font><code>true</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">e retornará </font></font><code>false</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">se o usuário pressionar 0.</font></font></p></body></html>

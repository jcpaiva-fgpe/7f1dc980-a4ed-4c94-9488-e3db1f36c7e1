        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Çoğu durumda, yöntemler, yürütmelerinin bir sonucu olarak bir miktar değer döndürür. </font><font style="vertical-align: inherit;">Bu değer, bazı değişkenlerde saklanabilir veya doğrudan çağırma yerine kullanılabilir.</font></font></p> 
        <h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Örnek</font></font></h3> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Kullanıcının devam edip etmeme isteğine karşılık gelen kodu bir fonksiyona çevireceğiz </font></font><code>elUsuarioQuieraSeguir()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, böylece metot kullanıcı tarafından yazılan değeri döndürür.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Bu yöntemdeki en önemli değişiklik, değiştiricinin </font></font><code>void</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">, dönüş değerinin veri türü ile değiştirilmesi gerektiğidir. </font><font style="vertical-align: inherit;">Şu anda, dönüş değeri kullanıcı tarafından yazılan sayı olacaktır, dolayısıyla dönüş değeri şöyle olacaktır </font></font><code>int</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">:</font></font></p> 
        <blockquote> 
         <p><code>&nbsp;&nbsp;&nbsp; static <strong>int</strong> elUsuarioQuieraSeguir() {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int seguir;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Scanner sc = new Scanner(System.in);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; seguir = sc.nextInt();<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; return seguir;<br> &nbsp;&nbsp;&nbsp; }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">main() yönteminin kodu şöyle görünür:</font></font></p> 
        <blockquote> 
         <p><code>&nbsp;&nbsp;&nbsp; public static void main(String[] args) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; int contador = 1, seguir = 1;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; while (<strong>elUsuarioQuieraSeguir() != 0</strong>) {<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // Invocamos el método cabecera<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; cabeceraTablaMultiplicar(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; tablaDeUnNumero(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // Invocamos el método pie<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; pieTablaMultiplicar(contador);<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; contador++;<br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; }<br> &nbsp;&nbsp;&nbsp; }</code></p> 
        </blockquote> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Egzersiz yapmak</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">İşlevi , yerine </font></font><code>elUsuarioQuieraSeguir()</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">a döndürecek şekilde </font><font style="vertical-align: inherit;">dönüştürün </font><font style="vertical-align: inherit;">: kullanıcı 0 dışında bir sayıya basarsa, işlev döner </font><font style="vertical-align: inherit;">ve </font><font style="vertical-align: inherit;">kullanıcı 0'a basarsa işlev döner.</font></font><code>boolean</code><font style="vertical-align: inherit;"></font><code>int</code><font style="vertical-align: inherit;"></font><code>true</code><font style="vertical-align: inherit;"></font><code>false</code><font style="vertical-align: inherit;"></font></p></body></html>

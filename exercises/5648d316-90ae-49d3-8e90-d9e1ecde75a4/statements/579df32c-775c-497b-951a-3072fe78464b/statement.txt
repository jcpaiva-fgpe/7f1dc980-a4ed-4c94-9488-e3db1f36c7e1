        <p><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Increment and decrement operators.</font></font></strong></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">The increment or decrement operations work as follows. </font><font style="vertical-align: inherit;">Suppose the variable a, of type int:</font></font></p> 
        <ul> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">a++ is equivalent to doing a = a + 1</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">++a is equivalent to doing a = a + 1</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">a-- is equivalent to doing a = a - 1</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">--a is equivalent to making a = a - 1</font></font></li> 
        </ul> 
        <p>&nbsp;</p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Is there a difference between a++ and ++a? </font><font style="vertical-align: inherit;">Obviously yes:</font></font></p> 
        <ul> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">a++ : first use the value of a in the corresponding operation, and then increment that value of a by 1</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">++a : first increments the value of a by 1 and then uses that incremented value in the corresponding operation</font></font></li> 
        </ul> 
        <p>&nbsp;</p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">The same thing happens with decrements:</font></font></p> 
        <ul> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">a-- : first use the value of a in the corresponding operation, and then decrement by 1 said value of a</font></font></li> 
         <li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">--a : first decrements the value of a by 1 and then uses that decremented value in the corresponding operation</font></font></li> 
        </ul>

        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Vi vill göra ett Java-program som jämför strängar och känner av om två strängar är lika.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Som vi vet, för att jämföra två strängar, som är objekt, kan vi inte använda likhetsoperatorn (==), eftersom denna operator jämför om båda objekten är samma (samma minnesplats), och inte om båda objekten eller strängarna innehåller samma värde samma text.</font></font></p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Du måste använda metoder i klassen String för att hantera strängar. </font><font style="vertical-align: inherit;">Du kan mycket väl behöva använda följande metoder:</font></font></p> 
        <ul> 
         <li><code>length()</code></li> 
         <li><code>charAt(n)</code></li> 
        </ul> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">För att läsa data från ingången kan du också mycket väl behöva använda följande metoder i klassen Scanner:</font></font></p> 
        <ul> 
         <li><code>nextInt()</code></li> 
         <li><code>nextLine()</code></li> 
        </ul> 
        <h3><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">INTRÄDE</font></font></strong></h3> 
        <p>La entrada comienza con un número indicando cuántos casos de prueba habrá que procesar. Cada uno de los casos estará formado por dos cadenas, cada una en una línea.</p> 
        <h3><strong>SALIDA</strong></h3> 
        <p>Por cada caso de prueba, el programa escribirá si las dos cadenas comparadas son iguales.</p> 
        <h3><strong>RESTRICCIONES</strong></h3> 
        <p>Deberá resolverse el problema diseñando una función que reciba dos cadenas de texto y devuelva un boolean en el que se indique si son iguales o no.</p> 
        <p>El programa principal, para cada uno de los casos,&nbsp;leerá los datos de entrada&nbsp;y usará la&nbsp;función&nbsp;para resolver el problema, imprimiendo posteriormente el resultado correspondiente.</p> 
        <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Du vet redan att i Java, för att lösa det här problemet med att jämföra strängar, finns det lika metoden. </font><font style="vertical-align: inherit;">Uppenbarligen handlar det om att du implementerar jämförelsealgoritmen, så du kan INTE använda metoden " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">equals</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> " eller " </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">compareTo</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> "-metoden, eller något liknande.</font></font></p></body></html>
